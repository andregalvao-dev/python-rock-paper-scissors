# python-rock-paper-scissors
# 🎮 Jokenpô em Python

Este é um projeto simples do clássico jogo **Jokenpô (Pedra, Papel e Tesoura)** desenvolvido em Python.

## 📌 Objetivo
Praticar lógica de programação, estruturas condicionais e interação com o usuário.

## 🚀 Como funciona
- O jogador escolhe entre pedra, papel ou tesoura
- O computador faz uma escolha aleatória
- O sistema compara as escolhas e define o vencedor

## 🧠 Tecnologias utilizadas
- Python 3

## ▶️ Como executar
1. Clone o repositório
2. Execute o arquivo `.py`
3. Jogue no terminal

## 🎯 Aprendizados
- Condicionais (`if/elif/else`)
- Biblioteca `random`
- Entrada de dados com `input()`
- Lógica de jogo
